# Activity Monitor

## Опис проєкту

Activity Monitor — це мобільний застосунок для моніторингу фізичної активності користувачів, розроблений у межах курсового проєкту.

Основною метою застосунку є надання користувачам можливості відстежувати власну фізичну активність, переглядати статистику та взаємодіяти з функціоналом мобільного застосунку, створеного на платформі Android.

У процесі розробки було реалізовано базові можливості застосунку, зокрема:
- створення інтерфейсу користувача;
- навігацію між екранами;
- роботу з даними;
- елементи авторизації;
- відображення інформації про активність користувача.

Проєкт створено з використанням Android Studio та мов програмування Java/Kotlin.

На даний момент застосунок є функціональним прототипом і ще перебуває на етапі вдосконалення. У майбутньому планується розширення функціоналу, оптимізація роботи застосунку та додавання нових можливостей для користувачів.

## Використані технології
- Android Studio  
- Java / Kotlin  
- XML (розмітка інтерфейсу)  
- HTML / CSS (для допоміжних сторінок або веб-компонентів проєкту)
